# UI for Zappos


### Build UI


```bash
npm i
npm i webpack -g
webpack
webpack --watch --watch-poll # for dev
```


### Build API


Set `API_URL` env to your API.


```bash
npm i
API_URL=https://b230e127-f938-4e94-99c9-c3e9f6a26fcf.dev.kuberlab.io/ node index.js # run server
```


Open `localhost:8888`.
